Pswitch embed values in a list according to a pattern of indices



Pswitch(list, which)


Pswitch chooses elements from the list by a stream of indices (which) and embeds them in the stream.

If the element is itself a pattern, it first completely embeds it before looking for the next index.


see also Pswitch1



(

var a, b;

a = Pseq([1, 2, 3], 2);

b = Pseq([65, 76]);

c = Pswitch([a, b, 800], Pseq([2, 2, 0, 1], inf));

x = c.asStream;

24.do({ x.next.postln; });

Ppoisson random values that follow a Poisson Distribution (positive integer values)


superclass: Pattern



Ppoisson(mean, length)

mean The mean of the distribution (defaults to 1)

length number of values produced  (default: inf)


// example


(

var a, c, w;

a = Ppoisson(1.0, inf);

c = a.asStream.nextN(500);

w = Window.new("Ppoisson", Rect(10, 10, 540, 800));

// plot the values

c.plot(bounds: Rect(10, 10, 520, 380), discrete: true, parent: w);

// a histogram of the values

c.histo(500).plot(bounds: Rect(10, 410, 520, 380), parent: w);

)

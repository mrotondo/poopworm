InRange tests if a signal is within a given range


inRange.ar(in, lo, hi)

InRange.kr(in, lo, hi)


If in is >= lo and <= hi output 1.0, otherwise output 0.0. Output is initially zero.


in - signal to be tested

lo - low threshold

hi - high threshold

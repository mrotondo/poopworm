Punop unary operator pattern


superclass: Pattern


Punop(operator, a)


Returns a stream that applies the unary operator to the stream values of the receiver .

Usually, this is the result of applying a unary operator (i.e. a method with one argument) to a pattern.


operator - operator to be applied

a - a pattern (or compatible pattern input)


examples of unary operators are: squared, sqrt, sin, tan ...


See also: Pnaryop, Pbinop, UnaryOpFunction



// example


(

var a;

a = Punop(\sqrt, Pseries(0, 1, 12));

a.asStream.all;

)


// this is the same as:

(

var a;

a = Pseries(0, 1, 12).sqrt;

a.asStream.all;

)


// some common cases:

Pseq([1, 2, 3]).squared;

Pseq([0.2, 0.5, 0.8]).coin;

Pwhite(-100, 100, inf).abs;
